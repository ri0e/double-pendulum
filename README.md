# Double Pendulum Simulator

A real-time, interactive **double pendulum simulator** built with **p5.js**. Explore chaotic motion and play with physics in your browser!

## Features

* **Adjustable Physics**: Gravity, rod lengths, and bob masses.
* **Mouse Interaction**: Drag bobs or push them with adjustable power.
* **Customizable Look**: Change colors of rods and bobs.
* **Trails**: See the path of both bobs in real-time.
* **Accurate Motion**: Uses **Runge-Kutta (RK4)** integration with damping for stability.

## How to Use

1. Open `index.html`.
2. Adjust settings from the panel.
3. Enable mouse interaction to drag or push bobs.
4. Watch chaotic motion unfold! >w<
